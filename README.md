# smart-meal-service
Тестовое задание на позицию Middle Fullstack C# Developer

## Описание

1. SmartMealService.Domain - библиотека с доменными моделями
2. SmartMealService.Services - библиотека, содержащая методы работы с backend сервером
3. SmartMealService.Infrastructure - библиотека для работы с базой данных
4. SmartMealService.CliClient - консольное приложение для получения блюд и регистрации заказов
5. SmartMealService.GuiClient - графическое прилоежение для чтения и редактирования переменных среды
6. SmartMealService.Tests - модульное и интеграционное тестирование

## Используемые технологии и библиотеки

- .NET 8
- gRPC
- WPF
- Entity Framework
- Serilog
- xUnit
