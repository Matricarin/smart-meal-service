# smart-meal-service
Тестовое задание на позицию Middle Fullstack C# Developer

## Описание

1. SmartMealService - библиотека для взаимодействия с backend сервером по HTTP
2. SmartMealService.ConsoleClient - консольное приложение для работы с базой данных
3. SmartMealService.WpfClient - графическое прилоежение для чтения и редактирования переменных среды
4. SmartMealService.Tests - модульное и интеграционное тестирование

## Используемые технологии и библиотеки

- .NET 8
- gRPC
- WPF
- Entity Framework
- Serilog
- xUnit
